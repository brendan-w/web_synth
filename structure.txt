=========================================================
Entry point
=========================================================

Main

	parameters
	----------------------------------
	Synth					Synth object
	Display					Display object
	Data					Data object


	functions
	----------------------------------
	init()					create synth/display/static objects
	frame()					animation loop (distributes events to synth and display)



=========================================================
Sound generation
=========================================================

Synth
	parameters
	----------------------------------
	Tracks[]				track list
	AudioCtx				audio context
	DestinationNode			audio destination
	BPM						int



	functions
	----------------------------------
	init()					setup/get audio context
	frame()					update the display
	addTrack()
	deleteTrack(num)
	start()
	stop()



	objects
	----------------------------------
	Track
		Pattern[][]			boolean
		OscillatorNodes[]	Oscillators
		gainNode			gain
		Tone				Wavetable (for oscillators)
		PatternButtons[][]	DOM objects
		startButton			DOM object
		stopButton			DOM object
		etc...

	
		next()				advances to next note
		setScale(scale[])	frequency list
		setTone(waveTable)	wavetable for oscillator
		setVolume()			set the gain


=========================================================
Static data
=========================================================

Data
	parameters
	----------------------------------
	Scales[]				list of Scale objects
	tones[]					list of Tone objects



	functions
	----------------------------------
	init()					compute wavetables



	objects
	----------------------------------
	Scale
		name				eg. major/minor/lydian etc...
		freq[]				table of frequencies for this scale

	Tone
		name
		real				real component
		imag				imaginary component
